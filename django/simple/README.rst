please put your simple first django setup here

Start::

	$ pip install django
	$ django-admin.py startproject simple
	$ cd simple
	$ python manage.py runserver
	
Clean-up::
	
	$ rm db.sqlite3
	
End::
	
	$ python manage.py runserver
	
Then go to http://127.0.0.1:8000/admin and login into server
